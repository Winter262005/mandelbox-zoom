
# Fractal genesis

### Content Warning

This project generates complex, high-frequency patterns and flashing light effects. It may not be suitable for individuals with photosensitive epilepsy or trypophobia.
<hr>

Weird little fractal universe generator.
Type in a word (anything like your name, etc) which the program takes in as parameters to generate fractal spaces unique to that word.

W/A/S/D = Move around <br>
Q/E = Up/Down <br>
Mouse = Look around

Have fun finding new worlds, Let me know if you find any wacky ones :)
